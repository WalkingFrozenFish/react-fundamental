Условная отрисовка. Можно использовать тернарный оператор для отрисовки компонентов. Стандартные циклы и условия ветвления не работают внутри jsx. В данном случае мы проверяем массив на наличие постов, если есть то отрисовываем их, иначе отрисовываем текст что постов нет.

Работает
{posts.length !== 0
    ? <PostList remove={removePost} posts={posts} title={"Языки программирования"} />
    : <h1 style={{ textAlign: "center", color: "green" }}>Посты не найдены!</h1>
}

Не работает
{if(posts.length !== 0) {
    <PostList remove={removePost} posts={posts} title={"Языки программирования"} />
} else {
    <h1 style={{ textAlign: "center", color: "green" }}>Посты не найдены!</h1>
}}
