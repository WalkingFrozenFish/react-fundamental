React - библиотека для создания пользовательских интерфейсов. Позволяет сосредоточиться на создании функционала, нежели чем на работе с DOM.

Библиотека создает легковесную копию DOM дерева, и фиксирует все изменения в дереве. Потом эти изменения сравнивает с исходным DOM деревом, и потом применяет эти изменения и перерисовывает дерево DOM.

Для установки библиотеки используется команда npx create-react-app app-name

Для запуска приложения используется команда npm start
